# Better Crunchyroll 

A Firefox extension that adds additional seetings to the Crunchyroll video player.

## Features

- **Configurable seek step**: Skip forwards/backwards by a custom number of seconds
- **Single-frame seeking**: Skip forwards/backwards a single frame using the `,` and `.` keys
- **Seek preview scaling**: Custom thumbnail preview size shown when hovering over the timeline
- **Hide recommendation carousel**: Automatically dismiss the end-of-episode recommendation banner
- **Fullscreen on double-click**: Enter and exit fullscreen by double-clicking on the player
